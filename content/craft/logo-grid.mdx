I reverse-engineered our production implementation from Navattic and rebuilt the same core interaction here.

The recreated version keeps these behaviors:

- Responsive 3x4 (mobile) and 6x2 (desktop) grid layout.
- Logos render in grayscale by default.
- A diagonal shimmer sweep loops across each cell with staggered delays.
- Hover pauses the shimmer across the grid.
- Logos with content metadata switch to full color, while the rest stay grayscale.
- Hover an eligible logo to scale it down and reveal either "See case study" or "See showcase".
- On mouse leave, the shimmer restarts after the hover fade transition.
